- 📦 **Data Structures:**  
  - Arrays / Lists
  - Linked Lists  
  - Stacks & Queues  
  - Trees & Binary Trees  
  - Graphs  
  - Hash Tables / Dictionaries  
  - Heaps & Priority Queues

- ⚡ **Algorithms:**  
  - Searching (Linear, Binary Search)  
  - Sorting (Bubble, Selection, Insertion, Merge, Quick Sort, etc.)  
  - Recursion  
  - Dynamic Programming  
  - Graph Algorithms (BFS, DFS, Dijkstra, etc.)  
  - Greedy Algorithms  
  - Backtracking  
  - Others...
